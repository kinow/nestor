Nestor QA — Open source test management tool

This is Nestor QA official web page. Nestor is an Open Source test management tool that was created with focus on simplicity, extensibility and integration with other tools.

The project is still in beta, but we already have a lot of ideas to put in practice. If you are interested in helping, feel free to browse the open issues or send us a message in our mailing list.

Check out the Getting Started guide for more.
